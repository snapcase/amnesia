# amnesia

A Crystal library for interacting with the [temp-mail API](https://temp-mail.org/en/api/).

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  amnesia:
    github: z64/amnesia
```


## Usage


```crystal
require "amnesia"
```
